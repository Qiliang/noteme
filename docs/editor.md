# 编辑器

## 功能
1. 支持markdown功能。使用 https://satteri.bruits.org/ 。
2. 左边是文件树，右边是编辑器。
3. 用CodeMirror 6来编辑文件。
4. 有编辑和预览两个模式切换。预览时编辑器被隐藏。
5. 可以粘贴图片，图片会保存到文档根目录下笔记旁的 `.assets/`，并插入到 markdown 中。
6. 密码标记（AES-256-GCM，noteme 自有格式，**不兼容** CryptoJS / OpenSSL `U2FsdGVkX1…`）：
   - 选中文本 → 工具栏「加密选区」→ 写入 `[passwd:密文]` 或 `[passwd:标签:密文]`。
   - 预览时显示解锁控件；输入口令后明文仅短暂显示（默认 30 秒），超时或刷新预览后重新锁定。
   - 可在「设置 → 密码标签」保存命名口令；带标签时解锁控件可预填口令（仍需手动解锁）。
